# 학생 유형별 데이터 분석 시뮬레이터 v2.5

교사가 교실에서 마주하는 다양한 학생 유형을 데이터로 진단하고 처방을 연습하는 실습용 에듀테크 웹앱입니다.

## 구성 파일

| 파일 | 역할 |
|------|------|
| `index.html` | 프론트엔드 웹앱 (단일 파일, 그대로 배포 가능) |
| `personas.js` | 12가지 학생 페르소나 데이터 |
| `Code.gs` | 백엔드 (Google Apps Script · 구글 시트 저장) |

## 주요 기능

- **12가지 학생 페르소나**: 무기력한 관찰자형, 초반 과열 방전형, 게임 몰입형, AI 의존형, 필분 폭발기형, 관계 지향 산만형, 고립된 우등생형, 보이지 않는 아이형, 디지털 거부형, 완벽주의 지연형, 정서적 불안형, 디지털 창작자형
- **입체적 데이터**: 정량 데이터(LMS 로그) + 정성 데이터(교사 관찰 메모) 결합 제시
- **4단계 지도 프로세스**: 데이터 분석 → 페르소나 진단 → 맞춤 처방 → 성장 확인
- **진단 대시보드**: 12명의 핵심 지표(성취도/정답률/자기주도성/동료관계)를 카드로 한눈에
- **구글 시트 저장**: 교사의 진단·처방 실습 결과가 시트에 자동 기록

## 배포 방법

### 1단계 — 구글 시트 백엔드 만들기 (선택)

1. [Google Sheets](https://sheets.google.com)에서 새 시트 생성
2. 메뉴 **확장 프로그램 → Apps Script** 클릭
3. 기본 코드를 지우고 `Code.gs` 전체를 붙여넣기
4. **배포 → 새 배포 → 웹 앱** 선택
   - 실행: **나**
   - 액세스: **모든 사용자**
5. 배포 후 받은 **웹 앱 URL** 복사

### 2단계 — 프론트엔드에 URL 연결

`index.html` 상단의 다음 줄에 URL을 붙여넣습니다.

```js
const GAS_URL = "여기에_웹앱_URL_붙여넣기";
```

> URL을 비워두면 구글 시트 저장 없이 "로컬 모드"로 동작합니다(실습 자체는 모두 가능).

### 3단계 — 웹앱 배포

`index.html`과 `personas.js`를 함께 다음 중 한 곳에 올리면 됩니다.

- **GitHub Pages**: 리포지토리에 업로드 후 Pages 활성화
- **Netlify / Vercel**: 폴더 드래그&드롭
- **구글 사이트 / 교내 LMS**: 정적 호스팅 영역에 업로드

## 사용 방법

1. 상단에 선생님 성함 입력
2. 좌측 명단 또는 대시보드에서 학생 선택
3. 데이터를 보고 유형을 진단 → 처방 선택 → 성장 결과 확인
4. 정답을 맞혀야 다음 단계로 진행되어 자연스럽게 학습됨
